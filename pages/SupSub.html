Demonstrating Subscript & Superscript Tags

Learn how to use subscript "sub" and superscript "sup" in chemical and mathematical formulas

Chemical Formulas

CO2

CO2 represents Carbon Dioxide.

H2O

H2O represents Water.

Mathematical Formulas

(A + B)2 = A2 + 2AB + B2

(A + B)2 expands into A2 + 2AB + B2.

(A - B)2 = A2 - 2AB + B2

(A - B)2 expands into A2 - 2AB + B2.
